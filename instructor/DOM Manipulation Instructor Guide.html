Context for Subject Material

DOM Manipulation is the most advanced concept covered in the Introductory Course.

It builds on many of the previously covered concepts, such as HTML, CSS, and JavaScript, to empower learners to create the kind of engaging user experience that can only be achieved through code.

It also prepares learners for the final module in the course, a 3 hour capstone project.

In keeping with the 'just enough to be dangerous' approach, the discussion is intentionally limited to a few of the most valuable concepts:

  • .querySelector/.querySelectorAll, which are more flexible than other DOM query methods such as .getElementById
  • .style, to change CSS rules dynamically
  • .setAttribute, which effectively encompasses the functionality of .className, .id, and a slew of other methods
  • .textContent for changing the text within an element (.append can't be used to update an element, .innerText isn't implemented as consistently across platforms)
  • .append for adding child elements (ignoring a slew of other options, including .appendChild, .innerHTML += for the sake of simplicity etc.)
  • A most cursory discussion of event listeners, avoiding discussions of bubbling vs capturing and event delegation until they can be covered more effectively in the frontend JavaScript course

Topic Prioritization

This is a dense module where all of the content can be directly applicable to building a frontend web application. With that in mind, the following topics are highest priority, as they are closely related to future concepts and develop skills from which the other concepts of this module can be extrapolated.

  1. Selecting elements with .querySelector
  2. Manipulating an element's CSS rules with .style
  3. Adding a event listener which changes the DOM when an element is clicked
